# OnAirNetsScraper
Desktop version of the website for extracting Netlogger data.

To install:
1.  git clone https://github.com/chrislrobson/OnAirNetsScraper.git
2.  cd OnAirNetsScraper
3.  tar xvfz OnAirNetScraper-18Jul2020-1638.tgz

To execute:
1.  python3 ./OnAirNetsScraper.py -Y -e
(note: "-e" will display errors such as errors from empty net listing returned by the Netlogger servers)
